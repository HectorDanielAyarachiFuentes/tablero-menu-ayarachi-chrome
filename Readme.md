## 🚀 Características Principales

### 🏠 Pantalla Principal — Tu centro de control

La pantalla principal combina todo lo que necesitás en cero clics: saludo personalizado según el momento del día, reloj en tiempo real, temperatura local y accesos a tus sitios favoritos.

![Pantalla principal v1](https://github.com/HectorDanielAyarachiFuentes/tablero-menu-ayarachi/blob/main/img-md/Inicioaplicacion.png?raw=true)

**Lo que ves al abrir una nueva pestaña:**
- 👋 **Saludo dinámico** — "Buenos días / Buenas tardes / Buenas noches" según la hora, personalizable con tu nombre
- 🕐 **Reloj en tiempo real** — soporte de formato 12h (AM/PM) o 24h con opción de mostrar segundos
- 🌡️ **Clima local** — temperatura y condición del tiempo detectada automáticamente o ingresada manualmente
- 🔍 **Buscador integrado** — con selector de motor de búsqueda (Google, DuckDuckGo, Bing, YouTube, Wikipedia, Ecosia y más)
- ⚡ **Accesos directos** — a tus sitios favoritos con íconos automáticos (favicon)

### 💾 Importar y Exportar Datos

Hacé backup de toda tu configuración o trasladala a otro dispositivo con un solo clic.

![Importar y exportar datos](https://github.com/HectorDanielAyarachiFuentes/tablero-menu-ayarachi/blob/main/img-md/Importardatoseexportar.png?raw=true)

Los datos se guardan en un archivo `data.json` que incluye:
- ✅ Todos tus accesos directos
- ✅ Carpetas organizadas
- ✅ Preferencias de apariencia
- ✅ Configuración del reloj y clima

---

### 🗑️ Papelera de Recuperación *(Nuevo en v2.0)*

¿Borraste un acceso directo sin querer? No hay problema — la papelera te permite **recuperar** elementos eliminados antes de que se borren definitivamente.

## 🛠️ Instalación

Para usar esta extensión como una página de nueva pestaña en Chrome/Brave/Edge:

**1.** 📥 Descargá o cloná este repositorio:
```bash
git clone https://github.com/HectorDanielAyarachiFuentes/tablero-menu-ayarachi.git
```

**2.** 🌐 Abrí tu navegador y ve a:
```
chrome://extensions
```

**3.** 🔧 Activá el **Modo desarrollador** (esquina superior derecha)

**4.** 📂 Hacé clic en **"Cargar descomprimida"** y seleccioná la carpeta `Menu-tab-2.0.0`

**5.** 🎉 ¡Listo! Abrí una nueva pestaña y disfrutá de tu tablero personalizado.

> **Compatibilidad:** Google Chrome · Brave · Microsoft Edge · Cualquier navegador basado en Chromium

---

## 🔒 Permisos requeridos

| Permiso | Motivo |
|---|---|
| `storage` | Guardar accesos, notas y preferencias localmente |
| `geolocation` | Detección automática del clima según tu ubicación |
| `bookmarks` | Importar marcadores del navegador como accesos |

> ⚠️ **Privacidad garantizada:** todos los datos se almacenan **localmente** en tu navegador. Nada se envía a servidores externos.
